# デザイン依頼文（外部AI向け）

## 判断：何を外部AIに投げ、何をコードで持つか
- **コードで持つ（外注しない）**：サイトのUI/レイアウト/テーマ（実装済み・Next.js + Tailwind。星空ダークテーマ）。再デザインAIに丸投げすると既存実装と乖離するため不要。
- **外部AIに投げる価値あり（サイトに今"無い"視覚資産）**：①ロゴ/ワードマーク ②OG/ヒーロー画像 ③ファビコン。これらは画像生成の領域でコードで作るものではない。

参考：各モダリティのベストは [[reference_external_ai_tools]]（ロゴ＝Recraft V4／雰囲気画像＝Midjourney/Flux）。

---

## ① ロゴ / ワードマーク（推奨：Recraft V4 — SVG・ブランド一貫性）
```
日本語の文字ロゴ「星空ノート」のワードマークを作成。
スタイル：ミニマルでモダン、夜空テーマ。深いネイビー背景に映える。
モチーフ：小さな星/星座の点と細い線を控えめにあしらう（やり過ぎない）。
配色：ベースは明るいオフホワイト〜淡いシアン(#6FD3E0)の発光、差し色に温かいオレンジ(#F0B357)を少量。
出力：横組みワードマーク＋単体アイコン（星モチーフ）の2パターン。SVG・透過背景。
NG：写実的な天体写真、にぎやかな装飾、既存ブランドの模倣。
```

## ② OG / ヒーロー画像（推奨：Midjourney or Flux — 雰囲気重視）
```
用途：WebサイトのOGP/ヒーロー背景。横長 1200x630（OG）と 1920x1080（ヒーロー）。
被写体：地平線の上に広がる満天の星空と淡い天の川。手前にシルエットの稜線や三脚を構える小さな人影（任意）。
トーン：深いネイビー〜藍のグラデーション、星はシアン白、地平付近にわずかな温かいオレンジの光。静かで詩的、過度に派手にしない。
構図：上半分〜中央に星空、下に余白（テキストを重ねられるよう右上または中央に空間を残す）。文字は入れない。
NG：実在の商標/ロゴ、過度なレンズフレア、非現実的なネオン。
```

## ③ ファビコン（Recraft or ②から切り出し）
```
星モチーフの単純なアイコン。16-512pxで潰れない単純形状。ネイビー背景＋シアンの星1つ＋小さな点。
```

「星空ノート」（星空撮影と天体観望の中立ガイド）のトップ・ヒーローセクションの

## 適用メモ
- 受け取ったロゴSVGは `app/` のヘッダー（現在テキスト「星空ノート」）に差し替え。
- OG画像は `app/opengraph-image.(png|tsx)` か layout の `openGraph.images` に設定（取得後）。
- ファビコンは `app/icon.svg` / `app/favicon.ico` に配置。
